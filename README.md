# aruco-poster-augmentation

## General Description
The ArucoMarker folder consists of input images named "0.jpg" to "10.jpg".These images have photographs of an aruco marker mounted on wall taken from various angles. 

The augmentation code is saved in Program.py file.

## Objectives

1. Arcuo marker detection
2. Use ArUco marker to estimate homography and warp a poster image so it fits naturally onto a wall's surface
   
## Dependencies

python 3.11.9
numpy 1.24.3
opencv-contrib-python 4.7.0.72
